Starter site for COMP1023 Lab 7 (Continuous Deployment with GitHub Pages).

Do not work in this repository. Click "Use this template" to create your own copy, following the instructions in the Lab 7 handout.

The site itself is three files: `_config.yml` sets the title and theme, `index.md` is the home page, and `another-page.md` is a second page. The visual theme is supplied by GitHub Pages, so there is no theme source to wade through.
