# Internal Documentation

Welcome to the internal documentation site. This page is written in Markdown and converted to HTML automatically, so keeping it up to date is a matter of editing text rather than hand-writing web pages.

## How this site gets published

Every push to the main branch starts a GitHub Actions workflow. That workflow builds these Markdown files into a static website and hands the result to GitHub Pages, which serves it. Nobody uploads anything by hand.

## Making a change

Edit this file, commit it, and then watch the Actions tab. Within a minute or two the change will be live on the public URL.
